# 自分専用暗記・一問一答アプリ（quiz-memorizer）

効率的な反復学習と弱点克服を目指して作成した、一問一答形式の暗記Webアプリです。  
市販のアプリでは実現しづらかった「間違えた問題の重点復習」や「スムーズなデータ管理」に特化して開発しました。
私自身のみが使うことを想定しているため、なるべくシンプルな作りにしています。

👉 **[アプリを実際に試す（GitHub Pages）](https://hisatoshi-ejiri-1.github.io/quiz-memorizer/)**

## 🌟 主な機能

* **効率的な学習ロジック**
  * **卒業システム**: 5回正解した問題は「卒業」となり、自動的に出題対象から除外。
  * **重複防止**: 直近5問で出題された問題は連続で出ない設計。
  * **柔軟な出題モード**: 10問 / 50問 / 100問 / 全問 から選択可能。
* **復習しやすい解答UI**
  * 回答後に手動で「次の問題へ」進む仕様にし、結果や解説・別解をじっくり確認可能。
  * 複数正解や表記揺れ（かな・カナ・漢字など）にも柔軟に対応。
* **直感的な問題管理**
  * Excelやスプレッドシートからテキストを一括ペースト登録可能。
  * キーワード検索、個別の編集・削除、正解数のリセット機能。
* **レスポンシブデザイン**
  * サイドバー（問題一覧）とメインエリア（出題・登録）の2カラム構成。スマホ閲覧時も快適に操作可能。

## 💡 開発の背景と工夫した点

「自分の学習ペースに完全に合わせた暗記ツールが欲しい」という思いから自作しました。

単に問題を解くだけでなく、**「定着した知識（5回正解）は引いていき、あやふやな問題に集中できる仕組み」**を取り入れることで学習効率を最大化しています。また、問題の追加や修正がストレスなく行えるよう、データの一括登録や管理画面の使いやすさにもこだわりました。
