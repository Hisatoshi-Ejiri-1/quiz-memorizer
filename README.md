## 📌 主な機能と特徴

- **重複なしの出題システム**
  - **10問 / 50問 / 100問 / 全問** モードを選択可能。
  - チャレンジ開始時に未完了問題をシャッフルし、設定した問題数が終わるまで重複なく1回ずつ出題。
- **記憶定着システム**
  - 正解するごとにカウントが増え、5回正解（または「覚えた！」ボタン押下）で「定着」扱いとなり、次回からの出題対象から除外。
- **柔軟な正解判定とシンプルな表示**
  - 「東京、とうきょう、トウキョウ」など複数の表記ゆれ正解パターンに対応。
  - 回答後の画面では、画面が見やすくなるよう基本回答（一番左の文字）のみを表示。
- **データ管理**
  - 1問ずつの登録・編集・削除、およびExcelからのコピー＆ペースト一括登録に対応。
  - ブラウザの `localStorage` に保存されるため、外部サーバー不要でローカル完結。
