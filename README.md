# 自分専用暗記・一問一答アプリ（quiz-memorizer）

効率的な反復学習と弱点克服を目指して作成した、一問一答形式の暗記Webアプリです。  
市販アプリにありがちな無駄な重複を排除し、スムーズなデータ管理と確実な知識定着に特化して開発しました。

👉 **[アプリを実際に試す（GitHub Pages）](https://hisatoshi-ejiri-1.github.io/quiz-memorizer/)**

## 🌟 主な機能

* **厳密なセット出題ロジック（重複排除）**
  * 指定した問題数（10問 / 50問 / 100問 / 全問）の中で**同じ問題が重複して出題されない**完全セット出題。
  * ランダムによる同じ問題の連続出題を防止し、選択したテンポで効率よく問題に回答可能。
* **卒業システムによる弱点集中**
  * 5回正解した問題は「覚えた（卒業）」と判定され、自動的に出題対象から除外。
* **視認性を追求した解答UI**
  * 回答後の「次の問題へ」画面では、混乱を防ぐため**基本回答（一番優先度が高い回答）のみを表示**するシンプルな設計。
  * 表記揺れ（かな・カナ・漢字など）や複数回答に対応しつつも、画面表示はすっきり整理。
* **直感的な問題管理**
  * Excelやスプレッドシートからテキストを一括ペースト登録可能。
  * 検索・編集・削除・進捗リセットができるサイドバー付き2カラムレイアウト。

## 💡 開発の背景と工夫した点

「自分の学習ペースに完全に合わせた、ストレスのない暗記ツールが欲しい」という思いから自作しました。

ランダム出題による無駄な重複をあえて排除し、**「1セット内で全問に1度ずつ触れられる出題ロジック」**や**「正解時に余計な別解情報を出さず基本回答に絞るUI」**など、学習時の集中力を削がないための細かなこだわりを詰め込んでいます。
